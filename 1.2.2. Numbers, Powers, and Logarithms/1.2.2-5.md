# 1.2.2-5

We just allow digits to be ${} 0..1 {}$ and don't allow infinite tail of ${} 1 {}$'s.

${} n + \frac{d_1}{2} + \frac{d^2}{4} + \cdots + \frac{d_k}{2^k} \leq x < n + \frac{d_1}{2} + \frac{d^2}{4} + \cdots + \frac{d_k}{2^k} + \frac{1}{2^k} {}$
