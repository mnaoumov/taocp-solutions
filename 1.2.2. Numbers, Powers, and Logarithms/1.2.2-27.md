# 1.2.2-27

${} x(1-\delta) \leq 10^n\cdot x_0' \leq x(1+\epsilon) {}$

Let's prove by induction:

${} (\frac{x(1-\delta)}{10^{n+\frac{b_1}{2^1}+\frac{b_2}{2^2}+\cdots+\frac{b_k}{2^k}}})^{2^k} \leq x_k' \leq (\frac{x(1+\epsilon)}{10^{n+\frac{b_1}{2^1}+\frac{b_2}{2^2}+\cdots+\frac{b_k}{2^k}}})^{2^k} {}$

${} \log'{x} = r_k:= n+\frac{b_1}{2^1}+\frac{b_2}{2^2}+\cdots+\frac{b_k}{2^k} {}$

${} k = 0 {}$, by definition

${} (k \implies k + 1) {}$

${} (\frac{x(1-\delta)}{10^{r_k}})^{2^{k+1}} \leq (x_k')^2 \leq (\frac{x(1+\epsilon)}{10^{r_k}})^{2^{k+1}} {}$

${} (\frac{x(1-\delta)}{10^{r_k}})^{2^{k+1}} (1-\delta) \leq (x_k')^2 (1-\delta) \leq y_{k+1} \leq (x_k')^2 (1+\epsilon) \leq (\frac{x(1+\epsilon)}{10^{r_k}})^{2^{k+1}} (1+\epsilon) {}$

${} 1 \leq y_{k+1} < 100 {}$

${} 1 \leq y_{k+1} \leq (\frac{x(1+\epsilon)}{10^{r_k}})^{2^{k+1}} (1+\epsilon) {}$

${} (\frac{x(1-\delta)}{10^{r_k}})^{2^{k+1}} (1-\delta) \leq y_{k+1} < 100 {}$

${} 0 \leq 2^{k+1} (\log_{10}{x} + \log_{10}{(1+\epsilon)} - r_k) + \log_{10}{(1+\epsilon)} {}$

${} \implies r_k \leq \log_{10}{x} + \log_{10}{(1+\epsilon)}(1+2^{-k-1}) \leq \log_{10}{x} + 2\log_{10}{(1+\epsilon)} {}$

${} 2^{k+1} (\log_{10}{x} + \log_{10}{(1-\delta)} - r_k) + \log_{10}{(1-\delta)} < 2 {}$

${} \implies r_k > \log_{10}{x} + \log_{10}{(1-\delta)}(1+2^{-k-1}) - \frac{1}{2^k} > \log_{10}{x} + 2\log_{10}(1-\delta) - \frac{1}{2^k} {}$
