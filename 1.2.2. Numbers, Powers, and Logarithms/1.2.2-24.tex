$n: 2^n \leq x < 2^{n+1}$

$x_0 = \frac{x}{2^n}$

$b_k = 0, x_k = x_{k-1}^2$ , if $x_{k-1}^2 < 2$

$b_k = 1, x_k = \frac{x_{k-1}^2}{2}$, if $x_{k-1}^2 \geq 2$

$1 \leq x_0 < 2$

$x_1 = \frac{x_0^2}{2^{b_1}}$

If $b_1=0 \implies x_1 = x_0^2 < 2$

If $b_1=1 \implies x_1 = \frac{x_0^2}{2} < \frac{2^2}{2} = 2$

$\implies 1 \leq x_1 < 2$

Then by induction $1 \leq x_k < 2$

$x_0 = \frac{x}{2^n}$

$x_1 = \frac{x_0^2}{2^{b_1}} = \frac{x^2}{2^{2n}\cdot2^{b_1}} = (\frac{x}{2^{n+\frac{b_1}{2}}})^2$

$x_2 = \frac{x_1^2}{2^b_2} = (\frac{x}{2^{n+\frac{b_1}{2} + \frac{b_2}{4}}})^4$

$x_k = (\frac{x}{2^{n+\frac{b_1}{2} + \frac{b_2}{4} + \cdots + \frac{b_k}{2^k}}})^{2^k}$

$1 \leq x_k < 2 \implies 1 \leq (\frac{x}{2^{n+\frac{b_1}{2} + \frac{b_2}{4} + \cdots + \frac{b_k}{2^k}}})^{2^k} < 2$

$1 \leq \frac{x}{2^{n+\frac{b_1}{2} + \frac{b_2}{4} + \cdots + \frac{b_k}{2^k}}} < 2^{\frac{1}{2^k}}$

$2^{n+\frac{b_1}{2} + \frac{b_2}{4} + \cdots + \frac{b_k}{2^k}} \leq x < 2^{n+\frac{b_1}{2} + \frac{b_2}{4} + \cdots + \frac{b_k}{2^k} + \frac{1}{2^k}}$

$\lg{x} = n+\frac{b_1}{2} + \frac{b_2}{4} + \cdots + \frac{b_k}{2^k} + \cdots$
