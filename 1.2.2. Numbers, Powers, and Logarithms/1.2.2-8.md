# 1.2.2-8

Choose ${} s \in \mathbb{N}_0: s^{m} \leq x < (s+1)^{m} {}$

${} (s + \frac{d_1}{10})^m \leq x < (s + \frac{d_1}{10} + \frac{1}{10})^m {}$

${} (s + \frac{d_1}{10} + \frac{d_2}{100})^m \leq x < (s + \frac{d_1}{10} + \frac{d_2}{100} + \frac{1}{100})^m {}$

${} \cdots {}$

${} (s + \frac{d_1}{10} + \frac{d_2}{100} + \cdots + \frac{d_k}{10^k})^m \leq x < (s + \frac{d_1}{10} + \frac{d_2}{100} + \cdots + \frac{d_k}{10^k} + \frac {1}{10^k})^m {}$

That's how we get the number ${} y = s + \frac{d_1}{10} + \frac{d_2}{100} + \cdots + \frac{d_k}{10^k} + \cdots {}$

Let ${} y_k := s + \frac{d_1}{10} + \frac{d_2}{100} + \cdots + \frac{d_k}{10^k} {}$

${} y_k^m \leq x < (y_k + \frac{1}{10^k})^m\ (1) {}$

${} y = \lim_{k \to \infty} y_k {}$

Take limit in ${} (1) {}$

${} y^m \leq x \leq y^m \implies y^m = x {}$

${} y < z \implies x = y^m < z^m \implies z \neq x {}$

${} y > z \implies x = y^m > z^m \implies z \neq x {}$

${} y {}$ - unique, such that ${} y^m = x {}$
