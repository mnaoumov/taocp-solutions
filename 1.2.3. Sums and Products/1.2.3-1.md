# 1.2.3-1

${} (a_i + a_{i+1} + \cdots + a_j) + (a_{j+1} + a_{j+2} + \cdots + a_k) = (a_i + a_{i+1} + \cdots + a_k) {}$

${} A := a_2 + \cdots + a_0 {}$

${} A + (a_1 + a_2) = a_2 + \cdots + a_2 = a_2 {}$

${} \implies A = -a_1 {}$
