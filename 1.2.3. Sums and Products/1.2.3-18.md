# 1.2.3-18

${} R(i) = \{ n {}$ is a multiple of ${} i \} {}$

${} S(i,j) = \{ 1 \leq j < i \} {}$

${} S'(j) = \{ \exists i: R(i) \land S(i,j) \} = \{ 1 \leq j < n \} {}$ (as for such ${} j: \exists i = n {}$, and for ${} j \geq n {}$, such ${} i {}$ doesn't exist)

${} R'(i,j) = \{ R(i) \land S(i,j) \} = \{ n {}$ is a multiple of ${} i, i > j \} {}$
