# 1.2.3-7

${} \sum_{R(j)}a_j = \lim_{n\to\infty} \sum_{\substack{R(j)\\0 \leq j < n}}a_j + \lim_{n\to\infty} \sum_{\substack{R(j)\\-n \leq j < 0}}a_j = \lim_{n\to\infty} \sum_{\substack{R(c-j)\\0 \leq c-j < n}}a_{c-j} + \lim_{n\to\infty} \sum_{\substack{R(c-j)\\-n \leq c-j < 0}}a_{c-j} = {}$

${} = \lim_{n\to\infty} \sum_{\substack{R(c-j)\\c-n < j \leq c}}a_{c-j} + \lim_{n\to\infty} \sum_{\substack{R(c-j)\\c < j \leq c + n}}a_{c-j} = \lim_{n\to\infty} \sum_{\substack{R(c-j)\\c-n+1 \leq j < c + 1}}a_{c-j} + \lim_{n\to\infty} \sum_{\substack{R(c-j)\\c + 1 \leq j < c + n + 1}}a_{c-j} =: A{}$

- ${} c + 1 = 0 \implies A = \lim_{n\to\infty} \sum_{\substack{R(c-j)\\-n \leq j < 0}}a_{c-j} + \lim_{n\to\infty} \sum_{\substack{R(c-j)\\0 \leq j < n}}a_{c-j} = \sum_{R(c-j)}a_{c-j} {}$
- ${} c + 1 > 0 \implies B := \sum_{\substack{R(c-j)\\0 \leq j < c + 1}} a_{c-j}, A = \lim_{n\to\infty} (\sum_{\substack{R(c-j)\\c-n+1 \leq j < c + 1}}a_{c-j} - B) + \lim_{n\to\infty} (B+\sum_{\substack{R(c-j)\\c + 1 \leq j < c + n + 1}}a_{c-j}) = \sum_{R(c-j)}a_{c-j} {}$
- ${} c + 1 < 0 \implies B := \sum_{\substack{R(c-j)\\c+1 \leq j < 0}} a_{c-j}, A = \lim_{n\to\infty} (\sum_{\substack{R(c-j)\\c-n+1 \leq j < c + 1}}a_{c-j} + B) + \lim_{n\to\infty} (-B + \sum_{\substack{R(c-j)\\c + 1 \leq j < c + n + 1}}a_{c-j}) = \sum_{R(c-j)}a_{c-j} {}$
