# 1.2.3-42

${} \sum_{1\leq i,j \leq n} \frac{-y+(x+ny)\delta_{ij}}{(x+ny)x} = \frac{1}{(x+ny)x}(-y \sum_{1\leq i,j \leq n} 1 + (x+ny) \sum_{1\leq i,j \leq n} \delta_{ij}) = \frac{1}{(x+ny)x}(-y n^2 + (x+ny) n) = \frac{nx}{(x+ny)x} = \frac{n}{x+ny} {}$

### v2

${} e = (1,1,\ldots,1)^T {}$

${} S = e^T C^{-1} e {}$

${} Ce=(x+ny)e \implies C^{-1}e = \frac{1}{x+ny}e {}$

${} S = e^Tc^{-1}e = \frac{1}{x+ny}e^Te = \frac{n}{x+ny} {}$
