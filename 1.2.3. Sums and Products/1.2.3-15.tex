$f_n(x) = 1x^1 + 2x^2 + \cdots + nx^n = \sum_{1 \leq j \leq n} jx^j = 1x^1 + \sum_{2 \leq j \leq n} jx^j = x + \sum_{1 \leq j - 1 \leq n - 1}(j-1+1)x^{j-1+1} = x + \sum_{1 \leq j \leq n - 1}(j+1)x^{j+1} =$

$= x + x \sum_{1 \leq j \leq n - 1} jx^j + \sum_{1 \leq j \leq n - 1}x^{j+1} = x + xf_{n-1}(x) + \sum_{0 \leq j - 1 \leq n - 2}x^{j-1+2} = x + xf_{n-1}(x) + x^2(\frac{x^{n-1} - 1}{x - 1}) = x + x(f_n(x)-nx^n) + \frac{x^2(x^{n-1} - 1)}{x-1}$

$(x-1)f_n(x) = -x + nx^{n+1} - \frac{x^2(x^{n-1}-1)}{x-1} = x(-1+nx^n - \frac{x(x^{n-1}-1)}{x-1}) = \frac{x((x-1)(-1+nx^n) - x(x^{n-1} - 1))}{x-1} =$

$= \frac{x}{x-1} (-x + 1 + nx^{n+1} - nx^n - x^n + x) = \frac{x(nx^{n+1}-(n+1)x^n+1)}{x-1}$

$f_n(x) = \frac{x(nx^{n+1}-(n+1)x^n+1)}{(x-1)^2}$

This works for $x \neq 1$

For $x=1: f_n(x) = n$
