# 1.2.3-24

Let ${} n {}$ - amount of ${} j: R(j) {}$

By condition ${} n {}$ is finite

${} n = 0 {}$

${} \log_b \prod_{R(j)} a_j = \log_b 1 = 0 = \sum_{R(j)} \log_b a_j {}$

${} (n \implies n + 1) {}$

Let ${} k {}$ be some value such as ${} R(k) {}$

${} S(j) := \{ j = k \}, T(j) := \{ R(j) \land j \neq k \} {}$

${} R(j) = \{ S(j) \lor T(j) \} \implies \prod_{R(j)} a_j = \prod_{S(j) \lor T(j)} a_j {}$

${} S(j) \land T(j) {}$ - none ${} \implies \prod_{S(j) \land T(j)} a_j = 1 {}$

${} \prod_{R(j)} a_j = \prod_{S(j) \lor T(j)} a_j \cdot \prod_{S(j) \land T(j)} a_j = \prod_{S(j)} a_j \cdot \prod_{T(j)} a_j = a_k \cdot \prod_{T(j)} a_j {}$

${} \log_b\prod_{R(j)} a_j = \log_b{a_k} + \log_b\prod_{T(j)} a_j {}$

${} \#\{ j: T(j) \} = \#\{ j: T(j) \} - 1 = n {}$

${} \implies \log_b\prod_{T(j)} a_j = \sum_{T(j)} \log_b{a_j} {}$

${} \log_b{a_k} = \sum_{S(j)} \log_b{a_j} {}$

${} \log_b\prod_{R(j)} a_j = \sum_{S(j)} \log_b{a_j} + \sum_{T(j)} \log_b{a_j} = \sum_{S(j) \lor T(j)} \log_b{a_j} + \sum_{S(j) \land T(j)} \log_b{a_j} = \sum_{R(j)} \log_b{a_j} + 0 = \sum_{R(j)} \log_b{a_j}{}$
