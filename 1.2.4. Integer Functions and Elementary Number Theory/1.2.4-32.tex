$\sum_{d \backslash n}\sum_{c \backslash d} f(c,d) = \sum_{c \backslash n}\sum_{d: c \backslash d, d\backslash n} f(c,d) =: A$

$d \backslash n \implies n = du$

$c \backslash d \implies d = cv$

$\implies n = cuv \implies \frac{n}{c} = uv$

$\implies v \backslash \frac{n}{c}$

$A = \sum_{c \backslash n}\sum_{v \backslash \frac{n}{c}, d = cv} f(c,d) =: A = \sum_{c \backslash n}\sum_{v \backslash \frac{n}{c}} f(c,cv)$
