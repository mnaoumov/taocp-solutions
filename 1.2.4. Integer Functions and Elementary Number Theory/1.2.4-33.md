# 1.2.4-33

### (a)

${} \lfloor \frac{1}{2}(n+m) \rfloor + \lfloor \frac{1}{2}(n-m+1) \rfloor =: A{}$

${} n+m = 2p+q, 0 \leq q \leq 1 {}$

${} A = \lfloor \frac{1}{2}(2p+q) \rfloor + \lfloor \frac{1}{2}(2p+q-2m+1) \rfloor = \lfloor p + \frac{q}{2} \rfloor + \lfloor p - m + \frac{q+1}{2} \rfloor = p + \lfloor \frac{q}{2} \rfloor + p - m + \lfloor \frac{q+1}{2} \rfloor = 2p - m + \lfloor \frac{q}{2} \rfloor + \lfloor \frac{q+1}{2} \rfloor =: B {}$

For ${} q = 0,1: \lfloor \frac{q}{2} \rfloor + \lfloor \frac{q+1}{2} \rfloor = q {}$

${} B = 2p-m+q=n+m-m=n {}$

### (b)

${} \lceil \frac{1}{2}(n+m) \rceil + \lceil \frac{1}{2}(n-m+1) \rceil =: A{}$

${} n+m = 2p+q, 0 \leq q \leq 1 {}$

${} A = \lceil \frac{1}{2}(2p+q) \rceil + \lceil \frac{1}{2}(2p+q-2m+1) \rceil = \lceil p + \frac{q}{2} \rceil + \lceil p - m + \frac{q+1}{2} \rceil = p + \lceil \frac{q}{2} \rceil + p - m + \lceil \frac{q+1}{2} \rceil = 2p - m + \lceil \frac{q}{2} \rceil + \lceil \frac{q+1}{2} \rceil =: B {}$

For ${} q = 0,1: \lceil \frac{q}{2} \rceil + \lceil \frac{q+1}{2} \rceil = q + 1{}$

${} B = 2p-m+q+1=n+m-m+1=n+1 {}$
