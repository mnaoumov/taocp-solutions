# 1.2.5-11

${} \mu(n,2) = \sum_{k > 0}{\lfloor{\frac{n}{2^k}}\rfloor} {}$

${} n = \sum_{i=1}^r 2^{e_i} {}$

${} \mu(n,2) = \sum_{k > 0}{\lfloor{\sum_{i=1}^r 2^{e_i-k}}\rfloor} = \sum_{k > 0}{\lfloor{\overbrace{\sum_{\substack{1 \leq i \leq r\\ e_i \geq k}} 2^{e_i-k}}^A + \overbrace{\sum_{\substack{1 \leq i \leq r\\ e_i < k}} 2^{e_i-k}}^B }\rfloor} {}$

${} A \in \mathbb{Z} {}$

${} 0 \leq B \leq \sum_{j=0}^{k-1} 2^{j-k} = 1 - 2^{-k} < 1 {}$

${} \mu(n,2) = \sum_{k > 0}{\lfloor{A+B}\rfloor} = \sum_{k > 0} A = \sum_{k>0} \sum_{\substack{1 \leq i \leq r\\ e_i \geq k}} 2^{e_i-k} = {}$

${} = \sum_{i=1}^r 2^{e_i} \sum_{\substack{k>0, k \leq e_i}}{2^{-k}} = \sum_{i=1}^r{2^{e_i}(1-2^{-e_i})} = \sum_{i=1}^r({2^{e_i}-1)} = n - r {}$
