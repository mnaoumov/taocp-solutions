# 1.2.1-2

The inductive step is wrong for ${} n = 1 {}$.

${} a^{(n-1)-1} = 1 {}$ is not explained. It seems the logic was to use ${} n - 1 < n {}$ so we could use inductive assumption, but for it to hold, ${} n - 1 {}$ has to be ${} \geq 1 {}$, which is not the case when ${} n = 1 {}$.
