# 1.2.7-8

 ${} A:=\sum_{k=1}^n H_k = (n + 1)H_n − n \approx (n + 1)(\ln{n} + \gamma + \frac{1}{2n}) - n = (n + 1)\ln{n} - (1-\gamma) n + (\gamma + \frac{1}{2}) {}$

 ${} B:=\sum_{k=1}^n \ln{k} \approx \ln n! = (n+\frac{1}{2}) \ln{n} - n + \ln{\sqrt{2\pi}} {}$

${} |A-B| \approx \gamma n + \frac{1}{2} \ln{n} + \gamma + \frac{1}{2} - \ln{\sqrt{2\pi}} {}$
