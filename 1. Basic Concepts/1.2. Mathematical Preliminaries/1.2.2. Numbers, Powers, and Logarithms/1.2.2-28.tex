Let binary computer's precision be \( p \implies x = \frac{a}{2^p}, y = \frac{c}{2^p}, a,c \in \mathbb{Z} \)

\( d_k := \frac{2^k+1}{2^k}, e_k := \lfloor2^p\log_b{d_k}\rfloor \)

L1. \( c \leftarrow 2^p, k \leftarrow 1 \)

L2. If \( a = 0, y = \frac{c}{2^p} \) - is the result. Stop.

L3. If \( a < e_k, k \leftarrow k + 1 \). Go to L3.

L4. \( a \leftarrow a - e_k, c \leftarrow c + \lfloor\frac{c}{2^k}\rfloor \). Go to L2.

\( c' = c + \lfloor\frac{c}{2^k}\rfloor = c + \frac{c}{2^k} - \{ \frac{c}{2^k} \} = cd_k - \{ \frac{c}{2^k} \} = cd_k (1 - \frac{\{ \frac{c}{2^k} \}}{cd_k}) \)

\( f := |x_0-x - \log_by| \)

\( f_0 = 0 \)

\( f'=|x_0-x' - \log_b{y'}| = |x_0-x + x - x' - \log_b{y} - \log_b{\frac{y'}{y}} | \leq f + | \frac{a-a'}{2^p} - \log_b{\frac{c'}{c}} | \leq f + | \frac{e_k}{2^p} - \log_b{(d_k(1 - \frac{\{ \frac{c}{2^k} \}}{cd_k}))} | \)

\( \frac{e_k}{2^p} = \frac{2^p\log_b{d_k} - \{2^p\log_b{d_k}\}}{2^p} = \log_b{d_k} - \frac{\{2^p\log_b{d_k}\}}{2^p} \)

\( f' \leq f + \frac{\{2^p\log_b{d_k}\}}{2^p} + |\log_b{(1 - \frac{\{ \frac{c}{2^k} \}}{cd_k})}| \leq f + \frac{1}{2^p} + \frac{|\ln(1 - \frac{\{ \frac{c}{2^k} \}}{cd_k})|}{|\ln{b}|} \leq f + \frac{1}{2^p} + \frac{1}{|\ln{b}|} \cdot \frac{\{ \frac{c}{2^k} \}}{cd_k} \leq f + \frac{1}{2^p} + \frac{1}{|\ln{b}|} \cdot \frac{1}{2^p \cdot 1} = f + \frac{1}{2^p} (1 + \frac{1}{|\ln{b}|}) \)

\( \implies f \leq \frac{1}{2^p} (1 + \frac{1}{|\ln{b}|}) \cdot C_{L4} \)
