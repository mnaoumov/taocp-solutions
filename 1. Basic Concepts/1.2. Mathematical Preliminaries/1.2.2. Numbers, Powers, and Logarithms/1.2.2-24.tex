\( n: 2^{n} \leq x < 2^{n+1} \)

\( x_{0} = \frac{x}{2^{n}} \)

\( b_{k} = 0, x_{k} = x_{k-1}^{2} \), if \( x_{k-1}^{2} < 2 \)

\( b_{k} = 1, x_{k} = \frac{x_{k-1}^{2}}{2} \), if \( x_{k-1}^{2} \geq 2 \)

\( 1 \leq x_{0} < 2 \)

\( x_{1} = \frac{x_{0}^{2}}{2^{b_{1}}} \)

If \( b_{1}=0 \implies x_{1} = x_{0}^{2} < 2 \)

If \( b_{1}=1 \implies x_{1} = \frac{x_{0}^{2}}{2} < \frac{2^{2}}{2} = 2 \)

\( \implies 1 \leq x_{1} < 2 \)

Then by induction \( 1 \leq x_{k} < 2 \)

\( x_{0} = \frac{x}{2^{n}} \)

\( x_{1} = \frac{x_{0}^{2}}{2^{b_{1}}} = \frac{x^{2}}{2^{2n}\cdot2^{b_{1}}} = {(\frac{x}{2^{n+\frac{b_{1}}{2}}})}^{2} \)

\( x_{2} = \frac{x_{1}^{2}}{2^{b}_{2}} = {(\frac{x}{2^{n+\frac{b_{1}}{2} + \frac{b_{2}}{4}}})}^{4} \)

\( x_{k} = {(\frac{x}{2^{n+\frac{b_{1}}{2} + \frac{b_{2}}{4} + \cdots + \frac{b_{k}}{2^{k}}}})}^{2^{k}} \)

\( 1 \leq x_{k} < 2 \implies 1 \leq {(\frac{x}{2^{n+\frac{b_{1}}{2} + \frac{b_{2}}{4} + \cdots + \frac{b_{k}}{2^{k}}}})}^{2^{k}} < 2 \)

\( 1 \leq \frac{x}{2^{n+\frac{b_{1}}{2} + \frac{b_{2}}{4} + \cdots + \frac{b_{k}}{2^{k}}}} < 2^{\frac{1}{2^{k}}} \)

\( 2^{n+\frac{b_{1}}{2} + \frac{b_{2}}{4} + \cdots + \frac{b_{k}}{2^{k}}} \leq x < 2^{n+\frac{b_{1}}{2} + \frac{b_{2}}{4} + \cdots + \frac{b_{k}}{2^{k}} + \frac{1}{2^{k}}} \)

\( \lg{x} = n+\frac{b_{1}}{2} + \frac{b_{2}}{4} + \cdots + \frac{b_{k}}{2^{k}} + \cdots \)
