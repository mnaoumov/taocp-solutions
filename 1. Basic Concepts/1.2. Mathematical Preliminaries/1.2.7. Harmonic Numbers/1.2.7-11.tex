\( \sum_{1<k\leq n} \frac{1}{k(k-1)}H_{k} = \sum_{2 \leq k < n + 1} \frac{1}{k(k-1)}H_{k} = \sum_{2 \leq k < n + 1} (\frac{1}{k-1} - \frac{1}{k}) H_{k} = \)

\( = - \sum_{2 \leq k < n + 1} (\frac{1}{k} - \frac{1}{k-1}) H_{k} =: A \)

\( a_{k} = \frac{1}{k-1}, b_{k} = H_{k} \)

\( A = -(a_{n+1}b_{n+1} - a_{2}b_{2} - \sum_{2\leq k < n + 1} a_{k+1}(b_{k+1}-b_{k})) = \)

\( = -\frac{1}{n} \cdot H_{n+1} + \frac{1}{1} \cdot H_{2} + \sum_{2\leq k < n + 1} \frac{1}{k}(H_{k+1}-H_{k}) = -\frac{H_{n+1}}{n} + \frac{3}{2} + \sum_{2\leq k < n+1}\frac{1}{k} \cdot \frac{1}{k+1} = \)

\( = -\frac{H_{n+1}}{n} + \frac{3}{2} + \sum_{2\leq k < n+1}(\frac{1}{k} - \frac{1}{k+1}) = -\frac{H_{n+1}}{n} + \frac{3}{2} + \frac{1}{2} - \frac{1}{n+1} = 2 - \frac{H_{n+1}}{n} - \frac{1}{n+1} = \)

\( = 2 - \frac{H_{n}}{n} - \frac{1}{n(n+1)} - \frac{1}{n+1} = 2 - \frac{H_{n}}{n} - \frac{1}{n} \)
