\( {(S_{1})}^{3} = T(\{\text{always}\}) = T(B_{1}) + T(B_{2}) + T(B_{3}) \)

\( B_{s} = \{ \#\{ i,j,k \} = s \} \)

\( T(B_{1}) = \sum_{i=0}^{n} a_{i}^{3} = S_{3} \)

\( T(B_{2}) = T(B_{2} \land \{i = j \neq k\}) + T(B_{2} \land \{j = k \neq i\}) + T(\overbrace{B_{2} \land \{k = i \neq j\}}^{B_{2}'}) = 3T(B_{2}') \)

\( T(B_{2}') = \sum_{\substack{0 \leq i \leq n \\ 0 \leq j \leq n \\ i \neq j }} a_{i}^{2}a_{j} = \sum_{\substack{0 \leq i \leq n \\ 0 \leq j \leq n }} a_{i}^{2}a_{j} - \sum_{\substack{0 \leq i \leq n \\ 0 \leq j \leq n \\ i = j }} a_{i}^{2}a_{j} = S_{1}S_{2} - S_{3} \)

\( T(B_{2}) = 3(S_{1}S_{2}-S_{3}) \)

\( T(B_{3}) = T(\overbrace{B_{3} \land \{ i < j < k \}}^{B_{3}'}) + T(B_{3} \land \{ j < i < k \}) + T(B_{3} \land \{ i < k < j \}) + T(B_{3} \land \{ j < k < i \}) + T(B_{3} \land \{ k < i < j \}) + T(B_{3} \land \{ k < j < i \}) = 6 T(B_{3}') \)

\( T(B_{3}') = \frac{1}{6}({(S_{1})}^{3} - 3S_{1}S_{2} + 2S_{3}) \)
