All \( \equiv \) will mean \( \equiv \pmod F_{n} \)

\( m = 4q+k, 0 \leq k \leq 3, 0 \leq r \leq n -1 \)

\( t = mn + r = 4qn + kn + r \)

\( q = 0 \)

\( k = 0: t = r, F_{t} = F_{r} \equiv F_{r} \)

\( k = 1: t = n + r \)

Let's prove \( F_{n+r} \equiv {(-1)}^{r+1}F_{n-r} \)

\( r = 0: F_{t}=F_{n} \equiv 0 \equiv {(-1)}^{0+1}F_{n-0} \)

\( r = 1: F_{t}=F_{n+1} \equiv F_{n} + F_{n-1} \equiv 0 + F_{n-1} = {(-1)}^{1+1}F_{n-1} \)

\( 2 \leq r \leq n - 1: F_{t} = F_{n+r} = F_{n+(r-1)} + F_{n+(r-2)} = {(-1)}^{(r-1)+1}F_{n-(r-1)} + {(-1)}^{(r-2)+1}F_{n-(r-2)} = \)

\( = {(-1)}^{r}F_{n-r+1} + {(-1)}^{r-1}F_{n-r+2} = {(-1)}^{r+1}(-F_{n-r+1} + F_{n-r+2}) = {(-1)}^{r+1}F_{n-r} \)

\( k = 2: t = 2n + r \)

Let's prove \( F_{2n+r} \equiv {(-1)}^{n}F_{r} \)

\( r = 0: F_{t}=F_{2n} \equiv 0 \equiv {(-1)}^{n}F_{0} \)

\( r = 1: F_{t}=F_{2n+1} = F_{2n} + F_{n+(n-1)} \equiv 0 + {(-1)}^{(n-1)+1}F_{n-(n-1)} = {(-1)}^{n}F_{1} \)

\( 2 \leq r \leq n - 1: F_{t} = F_{2n+r} = F_{2n+(r-1)} + F_{2n+(r-2)} = {(-1)}^{n}F_{r-1} + {(-1)}^{n}F_{r-2} = {(-1)}^{n}(F_{r-1} + F_{r-2}) = {(-1)}^{n}F_{r} \)

\( k = 3: t = 3n + r \)

Let's prove \( F_{3n+r} \equiv {(-1)}^{n+r+1}F_{n-r} \)

\( r = 0: F_{t}=F_{3n} \equiv 0 \equiv {(-1)}^{n+r+1}F_{n-0} \)

\( r = 1: F_{t}=F_{3n+1} = F_{3n} + F_{2n+(n-1)} \equiv 0 + {(-1)}^{n}F_{n-1} = {(-1)}^{n+1+1}F_{n-1} \)

\( 2 \leq r \leq n - 1: F_{t} = F_{3n+r} = F_{3n+(r-1)} + F_{3n+(r-2)} = {(-1)}^{n+(r-1)+1}F_{n-(r-1)} + {(-1)}^{n+(r-2)+1}F_{n-(r-2)} = \)

\( = {(-1)}^{n+r+1}(-F_{n-r+} + F_{n-r+2}) = {(-1)}^{n+r+1}F_{n-r} \)

Fully proven for \( q = 0 \)

\( (q \implies q + 1) \)

\( F_{4(q+1)n} \equiv 0 = F_{0} \)

\( F_{4(q+1)n+1} = F_{4(q+1)n} + F_{4(q+1)n - 1} \equiv F_{4q+3n+(n-1)} \equiv {(-1)}^{n+(n-1)+1}F_{n-(n-1)} = F_{1} \)

\( F_{4(q+1)n+r} \equiv F_{4qn+r}, r = 0, 1 \)

By Fibonacci property \( F_{4(q+1)n+r} \equiv F_{r} \forall r, 0 \leq r \leq 4n -1 \)

\( (q \implies q - 1) \)

\( F_{4(q-1)n + 3n + (n-1)} = F_{4qn - 1} = F_{4qn+1} - F_{4qn} \equiv F_{1} - F_{0} = F_{1} = {(-1)}^{n+(n-1)+1}F_{n-(n-1)} \equiv F_{4qn + 3n + (n-1)} \)

\( F_{4(q-1)n + 3n + (n-2)} = F_{4qn - 2} = F_{4qn} - F_{4qn-1} \equiv F_{0} - F_{1} = -F_{2} = {(-1)}^{n+(n-2)+1}F_{n-(n-2)} \equiv F_{4qn + 3n + (n-2)} \)

\( F_{4(q-1)n + r} \equiv F_{4qn + r}, r = 4n-1, 4n-2 \)

By Fibonacci property \( F_{4(q-1)n+r} \equiv F_{4qn + r} \forall r, 0 \leq r \leq 4n - 1 \)

Then we prove for \( q \in \mathbb{Z} \)
