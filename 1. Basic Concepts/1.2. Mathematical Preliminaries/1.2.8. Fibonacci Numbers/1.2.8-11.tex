\( \phi^{n} = F_{n} \phi + F_{n-1} \)

\( \widehat{\phi}^{n} = F_{n} \widehat{\phi} + F_{n-1} \)

\( n = 0 \)

\( F_{0} \phi + F_{-1} = 1 = \phi^{0} \)

\( F_{0} \widehat{\phi} + F_{-1} = 1 = \widehat{\phi}^{0} \)

\( (n \implies n + 1) \)

For \( x \in \{ \phi, \widehat{\phi} \}: x + 1 = x^{2}, x - 1 = \frac{1}{x} \)

\( F_{n+1} x + F_{n} = (F_{n} + F_{n-1})x + F_{n} = F_{n}(x+1) + F_{n-1} x = F_{n}x^{2} + F_{n-1}x = \)

\( = (F_{n}x + F_{n-1})x = x^{n} \cdot x = x^{n+1} \)

\( (n \implies n - 1) \)

\( F_{n-1} x + F_{n-2} = F_{n-1} x + (F_{n} - F_{n-1}) = F_{n} + F_{n-1} (x - 1) = F_{n} + \frac{F_{n-1}}{x} \)

\( = \frac{F_{n}x + F_{n-1}}{x} = \frac{x^{n}}{x} = x^{n-1} \)
