Let's prove by induction each \( n \geq 1 \) has representation \( P(n) \).

\( P(1) \)

\( n = 1 = F_{2}, 2 \gg 0 \)

Assume \( P(m) \) holds for \( 1 \leq m < n \)

Let
\[k_{1} \geq 2: F_{k_{1}} \leq n < F_{k_{1}+1} \tag{*}\label{eq:star}\]

If \( n = F_{k_{1}} \) --- representation

If \( n > F_{k_{1}} \implies m := n - F_{k_{1}}, 1 \leq m < n \)

\( P(m) \implies m = F_{k_{2}} + F_{k_{3}} + \cdots + F_{k_{r}}, k_{2} \gg k_{3} \gg \cdots \gg k_{r} \)

\( n = F_{k_{1}} + F_{k_{2}} + F_{k_{3}} + \cdots + F_{k_{r}} \)

We need to prove \( k_{1} \gg k_{2} \)

Assume \( k_{1} \not\gg k_{2} \implies k_{2} \geq k_{1} - 1 \)

\( n \geq F_{k_{1}} + F_{k_{2}} \geq F_{k_{1}} + F_{k_{1}-1} = F_{k_{1} + 1} \implies \) contradicts to~\eqref{eq:star} \( \implies \perp \)

\( \implies k_{1} \gg k_{2} \)

\( P(n) \) proven.

Now let's prove uniqueness

Let \( n \) be the smallest number with two representations.

Assume \( n = F_{k_{1}} + F_{k_{2}} + \cdots + F_{k_{r}} = F_{l_{1}} + F_{l_{2}} + \cdots + F_{l_{s}} \)

\( k_{1} \gg k_{2} \gg \cdots \gg k_{r} \)

\( l_{1} \gg l_{2} \gg \cdots \gg l_{s} \)

\( n = F_{k_{1}} + F_{k_{2}} + \cdots + F_{k-r} \leq F_{k_{1}} + F_{k_{1} - 2} + F_{k_{1} - 4} + \cdots + F_{k_{1} - 2r + 2} \)

\( n + F_{k-1-2r+1} \leq F_{k_{1}} + F_{k_{1} - 2} + F_{k_{1} - 4} + \cdots + (F_{k_{1} - 2r + 2} + F_{k_{1} - 2r + 1}) = \)

\( = F_{k_{1}} + F_{k_{1} - 2} + F_{k_{1} - 4} + \cdots + (F_{k_{1} - 2r + 4} + F_{k_{1} - 2r + 3}) = \cdots = F_{k_{1} + 1} \)

\( F_{k_{1}} \leq n < F_{k_{1} + 1} \)

\( F_{l_{1}} \leq n < F_{l_{1} + 1} \)

\( \implies k_{1} = l_{1} \)

\( m = n - F_{k_{1}} = n - F_{l_{1}} \)

\( m = F_{k_{2}} + F_{k_{3}} + \cdots + F_{k_{r}} = F_{l_{2}} + F_{l_{3}} + \cdots + F_{l_{s}} \)

\( m = 0 \lor 1 \leq m < n \)

In both cases, the representation of \( m \) is unique \( \implies \) for \( n \) it's unique.
