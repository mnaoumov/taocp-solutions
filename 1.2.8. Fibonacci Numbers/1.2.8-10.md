# 1.2.8-10

${} F_{n} = \frac{1}{\sqrt{5}}(\phi^n - \widehat{\phi}^n) = \frac{\phi^n}{\sqrt{5}} - \frac{\widehat{\phi}^n}{\sqrt{5}} {}$

${} \widehat{\phi} = \frac{1-\sqrt{5}}{2} < 0 {}$

${} \widehat{\phi}^{2k} > 0 {}$

${} \widehat{\phi}^{2k+1} < 0 {}$

${} F_{2k} < \frac{\phi^{2k}}{\sqrt{5}} {}$

${} F_{2k+1} > \frac{\phi^{2k+1}}{\sqrt{5}} {}$
