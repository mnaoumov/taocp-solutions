$F(x) = \sum_{k=0}^n F_kx^k$

$F(x)(1-x-x^2) = \sum_{k=0}^n F_kx^k - \sum_{k=1}^{n+1} F_{k-1}x^k - \sum_{k=2}^{n+2} F_{k-2}x^k =$

$= 0 + x + \sum_{k=2}^n(F_k - F_{k-1}-F_{k-2})x^k - F_{n}x^{n+1} - F_{n-1}x^{n+1} - F_nx^{n+2} = x - (F_n+F_{n-1})x^{n+1} - F_nx^{n+2} =$

$= x - F_{n+1}x^{n+1} - F_nx^{n+2}$

$F(x) = \frac{x-F_{n+1}x^{n+1}-F_nx^{n+2}}{1-x-x^2}$

When $x = \frac{1}{\phi}$ or $x = \frac{1}{\hat\phi}: 1-x-x^2 = 0$

$F(x) = \frac{(x-F_{n+1}x^{n+1}-F_nx^{n+2})'}{(1-x-x^2)'} = \frac{1-(n+1)F_{n+1}x^n -(n+2)F_nx^{n+1}}{-1-2x} = \frac{-1+(n+1)F_{n+1}x^n+(n+2)F_nx^{n+1}}{1+2x}$
