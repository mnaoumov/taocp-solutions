# 1.2.8-31

${} F_{-2n} = (-1)^{2n+1} F_{2n} = -F_{2n} {}$

${} \phi^{-2n} = F_{-2n}\phi + F_{-2n-1} = -F_{2n}\phi + F_{-2n-1} {}$

${} F_{2n}\phi = (F_{-2n-1} - 1) + (1 - \phi^{-2n}) {}$

${} F_{2n}\phi \mod 1 = (1 - \phi^{-2n}) \mod 1 {}$

${} 0 < 1 - \phi^{-2n} < 1 {}$

${} F_{2n}\phi \mod 1 = 1 - \phi^{-2n} {}$

${} F_{-(2n+1)} = (-1)^{2n+2} F_{2n+1} = F_{2n+1} {}$

${} \phi^{-(2n+1)} = F_{-(2n+1)}\phi + F_{-(2n+1)-1} = F_{2n+1}\phi + F_{-2n-2} {}$

${} F_{2n+1}\phi = -F_{-2n-2} + \phi^{-2n-1)}{}$

${} F_{2n+1}\phi \mod 1 = \phi^{-2n-1} \mod 1 {}$

${} 0 < \phi^{-2n-1} < 1 {}$

${} F_{2n+1}\phi \mod 1 = \phi^{-2n-1} {}$
