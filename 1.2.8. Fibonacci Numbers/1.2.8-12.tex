$F(x) = \sum_{n\geq 0}F_nx^n = \frac{x}{1-x-x^2}$

$\mathcal{F}_0 = 0, \mathcal{F}_1 = 1, \mathcal{F}_{n+2} = \mathcal{F}_{n+1} + \mathcal{F}_{n} + F_n$

$\mathcal{F}(x) = \sum_{n\geq 0} \mathcal{F}_n x^n = x + \sum_{n \geq 0} \mathcal{F}_{n+2}x^{n+2} = x + \sum_{n \geq 0} (\mathcal{F}_{n+1} + \mathcal{F}_{n} + F_n)x^{n+2} =$

$= x + x\sum_{n \geq 0}\mathcal{F}_{n+1}x^{n+1} + x^2\sum_{n \geq 0}\mathcal{F}_{n}x^{n} + x^2\sum_{n \geq 0}F_nx^n =$

$= x + x\sum_{n \geq 1}\mathcal{F}_{n}x^{n} + x^2\sum_{n \geq 0}\mathcal{F}_{n}x^{n} + x^2\sum_{n \geq 0}F_nx^n =$

$= x + x (\mathcal{F}(x) - \mathcal{F}_0x^0) + x^2\mathcal{F}(x)+x^2F(x) =$

$= \mathcal{F}(x)(x+x^2)+x+x^2F(x)$

$(1-x-x^2)\mathcal{F(x)} = \frac{x}{1-x-x^2} + x^2F(x)$

$\mathcal{F(x)} = F(x) + \frac{x^2F(x)}{1-x-x^2}$

$\mathcal{F(x)} = F(x) + x(F(x))^2$

$\mathcal{F}_n = [x^n]\mathcal{F(x)} = [x^n]F(x) + [x^n]x(F(x))^2 =$

$= F_n + [x^{n-1}](F(x))^2 = F_n + \sum_{k=0}^{n-1}F_kF_{n-k} =$

$= F_n + \frac{1}{5}(n-2)F_{n-1} + \frac{2}{5}(n-1)F_{n-2} =: A$

$F_{n-1} = F_{n+1} - F_{n}$

$F_{n-2} = F_{n} - F_{n-1} = 2F_{n} - F_{n+1}$

$A = F_n + \frac{1}{5}(n-2)(F_{n+1} - F_{n}) + \frac{2}{5}(n-1)(2F_{n} - F_{n+1}) =$

$= \frac{1}{5}(F_{n+1}((n-2)-2(n-1)) + F_n(5-(n-2)+4(n-1))) =$

$= \frac{1}{5}(-nF_{n+1} + F_n(3+3n)) = \frac{3}{5}(n+1)F_n - \frac{1}{5}nF_{n+1}$
