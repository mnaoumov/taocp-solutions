## 1

## 2

We need to prove that ${} \forall x \geq 0 \exists m \in \mathbb{Z} \forall k \geq m \exists! \alpha_k \in \{ 0, 1 \}: x = \sum_{k}\alpha_k \phi^{-k} {}$

${} \exists m \in \mathbb{Z} \forall k < m: \alpha_k = 0 \quad (1) {}$

${} \nexists k: \alpha_k=\alpha_{k+1} = 1 \quad (2) {}$

${} \nexists n \forall l\geq 0: \alpha_{n+2l} = 1, \alpha_{n+2l + 1} = 0 \quad (3) {}$

## Existence

Let ${} x \geq 0 {}$

If ${} x = 0 {}$, we take ${} \alpha_k = 0, k \in \mathbb{Z} {}$

If ${} x > 0 \exists m \in \mathbb{Z}: \phi^{-m} \leq x < \phi^{-m + 1} {}$. Let ${} \alpha_m := 1, \alpha_k := 0, k < m \implies (1) {}$

${} x_n := \sum_{k \leq n} \alpha_k \phi^{-k} {}$

${} \forall k<m: x_k=0, x_m = \phi^{-m} {}$

${} \forall k \geq m + 1: \alpha_{k} := [x-x_{k-1} \geq \phi^{-k}] {}$

Let's prove ${} \forall k \geq m: 0 \leq x - x_k < \phi^{-k} \quad P(k){}$

${} P(m): \phi^{-m} - \phi^{-m} \leq x - x^m < \phi^{-m+1} - \phi^{-m} = \phi^{-m}(\phi - 1) {}$

${} \phi^2 = \phi + 1 \implies \phi = 1 + \phi^{-1} \implies \phi - 1 = \phi^{-1} {}$

${} 0 \leq x - x_m < \phi^{-m}\cdot\phi^{-1} < \phi^{-m} {}$

${} (k \implies k + 1) {}$

Assume ${} P(k) \implies 0 \leq x - x_k < \phi^{k} {}$

${} P(k+1): x - x_{k+1} = x-x_k - \alpha_{k+1}\phi^{k+1} {}$

If ${} \alpha_{k+1} = 0 \implies x - x_k < \phi^{-(k+1)} {}$

${} x_{k+1} = x_k \implies 0 \leq x - x_{k+1} = x - x_k < \phi^{-(k+1)}  {}$

If ${} \alpha_{k+1} = 1 \implies x - x_k \geq \phi^{-(k+1)} {}$

${} x - x_{k+1} = x - x_k - \phi^{-(k+1)} \geq 0 {}$

${} x - x_{k+1} = x - x_k - \phi^{-(k+1)} < \phi^{-k} - \phi^{-(k+1)} = \phi^{-(k+1)} (\phi - 1) = {}$

${} = \phi^{-(k+1)} \cdot \phi^{-1} < \phi^{-(k+1)} {}$

${} \implies P(k+1) {}$

${} \implies \lim_{k \to \infty} x_k = x {}$

${} x = \sum_{k} \alpha_k \phi^{-k} {}$

Now let's prove ${} (2) {}$

Assume ${} k = \min \{ s: \alpha_s = \alpha_{s+1} = 1 \} {}$

As ${} k {}$ is minimal ${} \implies \alpha_{k-1} = 0 {}$

${} x - x_{k-2} < \phi^{-(k-1)} \quad (4) {}$

${} x_{k-1} = x_{k-2} {}$

${} \alpha_k = 1 \implies x - x_{k-1} \geq \phi^{-k} {}$

${} x_k = x_{k-1} + \phi^{-k} = x_{k-2} + \phi^{-k} {}$

${} \alpha_{k+1} = 1 \implies x - x_{k} \geq \phi^{-(k+1)} {}$

${} x - (x_{k-2} + \phi^{-k}) \geq \phi^{-(k+1)} {}$

${} x - x_{k-2} \geq \phi^{-k} + \phi^{-(k+1)} = \phi^{-(k+1)}(\phi + 1) =  \phi^{-(k+1)} \cdot \phi^2 = \phi^{-(k-1)} {}$

Contradicts to ${} (4) \implies \perp \implies (2) {}$

Now let's prove ${} (3) {}$

Assume ${} \exists n \forall l\geq 0: \alpha_{n+2l} = 1, \alpha_{n+2l + 1} = 0 {}$

${} \alpha_n = 1 \overset{(2)}{\implies} \alpha_{n-1} = 0 {}$

${} x - x_{n-2} < \phi^{-(n-1)} \quad (5) {}$

${} x_{n-1} = x_{n-2} {}$

${} x = x_{n-1} + \sum_{l \geq 0}\phi^{-(n+2l)} = x_{n-2} + \frac{\phi^{-n}}{1-\phi^{-2}} {}$

${} \phi^2 = \phi + 1 \implies 1 = \phi^{-1} + \phi^{-2} \implies 1 - \phi^{-2} = \phi^{-1} {}$

${} x = x_{n-2} + \phi^{-(n-1)} {}$

${} x - x_{n-2} = \phi^{-(n-1)} {}$

Contradicts to ${} (5) \implies \perp \implies (3) {}$

## Uniqueness

Assume there are different expansions

Let ${} x = \sum_k \alpha_k \phi^{-k} = \sum_k \beta_k \phi^{-k} {}$

${} m = \min \{k: \alpha_k \neq \beta_k \} {}$

For clarity, ${} \alpha_m = 1, \beta_m = 0 {}$

${} \forall k<m: \alpha_k = \beta_k \implies x_{m-1} = \sum_{k<m} \alpha_k \phi^{-k} = \sum_{k<m} \beta_k \phi^{-k} {}$

${} \implies y = x - x_{m-1} = \sum_{k \geq m} \alpha_k \phi^{-k} = \sum_{k \geq m} \beta_k \phi^{-k} {}$

${} y \geq \alpha_{m} \phi^{-m} = \phi^{-m} \quad (6) {}$

Let ${} B := \{ k \geq m + 1: \beta_k = 1 \} = \{ k_1, k_2, \ldots, k_r \}, 0 \leq r \leq \infty, k_1 < k_2 < \cdots < k_r {}$

${} y = \sum_{j = 1}^r \phi^{-k_j} {}$

${} \beta_{k_j} = 1 \implies \beta_{k_j+1} = 0 \implies k_{j+1} \geq k_j + 2 \implies k_j \geq m + 2j - 1 {}$

${} y = \sum_{j = 1}^r \phi^{-k_j} \leq \sum_{j = 1}^r \phi^{-(m-1+2j)} {}$

If ${} r < \infty \implies y < \sum_{j = 1}^\infty \phi^{-(m-1+2j)} = \frac{\phi^{-(m+1)}}{1-\phi^{-2}} = \phi^{-m} {}$

If ${} r = \infty, (3) \implies y < \sum_{j = 1}^\infty \phi^{-(m-1+2j)} = \phi^{-m} {}$

In both cases ${} y < \phi^{-m} {}$, contradicts to ${} (6) \implies \perp {}$

${} \implies {}$ uniqueness.

${} \forall n \leq L_3 = 4: n \in A_{-2,2} {}$

${} (p \implies p + 1) {}$
