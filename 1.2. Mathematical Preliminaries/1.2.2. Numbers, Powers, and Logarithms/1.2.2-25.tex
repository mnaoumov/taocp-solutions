\subsubsection*{v2}

Let binary computer's precision be \( p \implies x = \frac{a}{2^p}, y = \frac{c}{2^p}, z = \frac{d}{2^p}, a,c,d \in \mathbb{Z} \)

Let \( C_{L4} \) - count of how many times step L4 was executed.

L1: \( c \leftarrow 0, d \leftarrow \lfloor \frac{a}{2} \rfloor, k \leftarrow 1 \)

L2: If \( a = 2^p \), stop.

L3: If \( a - d < 2^p, d \leftarrow \lfloor\frac{d}{2}\rfloor, k \leftarrow k + 1 \)

L4: \( a \leftarrow a - d, d \leftarrow \lfloor\frac{a}{2^k}\rfloor, c \leftarrow c + \lfloor2^p\log_b\frac{2^k}{2^k-1}\rfloor, C_{L4} \leftarrow C_{L4} + 1 \)

\( e_k := \frac{2^{k}}{2^{k}-1}, f_k := \lfloor2^p\log_be_k\rfloor \)

\( g := \frac{1}{2^p}(1 + \frac{1}{|\ln b|}) \)

A1 (after L1): \( a_0 = a, c = 0, d = \lfloor\frac{a}{2}\rfloor, k = 1, a - 2d < 2^p, C_{L4} = 0, h = |\log_b\frac{a_0}{a}-\frac{c}{2^p}| \leq C_{L4} \cdot g \)

A2 (before L2) \( : a \geq 2^p, d = \lfloor\frac{a}{2^k}\rfloor, a - 2d < 2^p, h \leq C_{L4} \cdot g \)

A3 (if L2 succeeds): \( a = 2^p, |\log_b{x_0} - y| \leq C_{L4} \cdot g \)

A4 (before L3): \( a > 2^p, d = \lfloor\frac{a}{2^k}\rfloor, a - 2d < 2^p \)

A5 (before L4): \( a - d \geq 2^p, d = \lfloor\frac{a}{2^k}\rfloor, a - 2d < 2^p \)

Proof:

A1: \( a-2d=a-2\lfloor\frac{a}{2}\rfloor = 2 \{\frac{a}{2}\} < 2 \leq 2^p \)

A2 (after L1): \( d = \lfloor\frac{a}{2^1}\rfloor \)

A2 (after L4): Let's use indices with prime (\texttt{'}) for values after transform \( a'=a-d, d'=\lfloor\frac{a'}{2^k}\rfloor, c'=c + f_k \)

\( a'=a-d \geq 2^p \)

\( d'=\lfloor\frac{a'}{2^k}\rfloor \)

\( a-2d'=a-2\lfloor\frac{a}{2^{k'}}\rfloor = a-2(\frac{a}{2^{k'}} - \{\frac{a}{2^{k'}}\}) = a-\frac{a}{2^{k'-1}} - 2\{\frac{a}{2^{k'}}\} = a-\frac{a}{2^k} - 2\{\frac{a}{2^{k'}}\} = a - \lfloor\frac{a}{2^k}\rfloor + \{\frac{a}{2^k}\} - 2\{\frac{a}{2^{k+1}}\} = \)

\( = a - d + \{\frac{a}{2^k}\} - 2\{\frac{a}{2^{k+1}}\} = a - d + \lfloor\{\frac{a}{2^k}\} - 2\{\frac{a}{2^{k+1}}\}\rfloor \leq a - d < 2^p \)

A5 \( \implies a' = a - d \geq 2^p, d' = \frac{a'}{2^k} \)

\( a' - 2d'=a-d-2\cdot\frac{a-d}{2^k} = (a-d)(1 - \frac{2}{2^k}) =: B \)

A5 \( \implies 2^k \leq \frac{a}{d} \)

\( B \leq (a-d)(1-\frac{2d}{a}) = \frac{a-d}{a}\cdot(a-2d) < 1 \cdot 2^p = 2^p \)

\( h' = |\log_b\frac{a_0}{a'}-\frac{c'}{2^p}| = |\log_b\frac{a_0}{a}-\log_b\frac{a'}{a}-\frac{c}{2^p} - \frac{f_k}{2^p}| \leq h + |\log_b\frac{a'}{a} + \frac{f_k}{2^p} | \)

\( a'=a-\lfloor\frac{a}{2^k}\rfloor = a - \frac{a}{2^k} + \{ \frac{a}{2^k} \} = \frac{a}{e_k} + \{ \frac{a}{2^k} \} = \frac{a}{e_k}(1 + \frac{e_k}{a}\cdot \{ \frac{a}{2^k} \}) \)

\( \log_b\frac{a'}{a} = -\log_b{e_k} + \log_b{(1 + \frac{e_k}{a}\cdot \{ \frac{a}{2^k} \})} \)

\( \frac{f_k}{2^p} = \frac{\lfloor2^p\log_b{e_k}\rfloor}{2^p} = \frac{2^p\log_b{e_k} - \{2^p\log_b{e_k}\}}{2^p} = \log_b{e_k} - \frac{\{2^p\log_b{e_k}\}}{2^p} \)

\( \log_b\frac{a'}{a} + \frac{f_k}{2^p} = \log_b{(1 + \frac{e_k}{a}\cdot \{ \frac{a}{2^k} \})} - \frac{\{2^p\log_b{e_k}\}}{2^p} \)

\( h' \leq h + |\log_b{(1 + \frac{e_k}{a}\cdot \{ \frac{a}{2^k} \})}| + \frac{\{2^p\log_b{e_k}\}}{2^p} \)

\( |\log_b{(1 + \frac{e_k}{a}\cdot \{ \frac{a}{2^k} \})}| = \frac{|\ln{(1 + \frac{e_k}{a}\cdot \{ \frac{a}{2^k} \})}|}{|\ln{b}|} \leq \frac{\frac{e_k}{a}\cdot \{ \frac{a}{2^k} \}}{|\ln{b}|} =: A \)

Let \( a = 2^k\cdot q + r, 0 \leq r \leq 2^k - 1 \implies \frac{a}{2^k} = q + \frac{r}{2^k} \implies \{ \frac{a}{2^k} \} \leq \frac{2^k-1}{2^k} = \frac{1}{e_k} \)

\( a \geq 2^p \)

\( A \leq \frac{1}{2^p |\ln b| } \)

\( \frac{\{2^p\log_b{e_k}\}}{2^p} \leq \frac{1}{2^p} \)

\( h' \leq h + \frac{1}{2^p} (1 + \frac{1}{|\ln b|}) \leq (C_{L4} + 1) \cdot g = C'_{L4} \cdot g \)

A3: \( h = |\log_b\frac{x_0\cdot2^p}{2^p} - y| = |\log_b{x_0} - y| \leq C_{L4} \cdot g \)

A4 (after L2): obvious

A4 (after L3): \( d' = \lfloor\frac{d}{2}\rfloor = \lfloor\frac{\lfloor\frac{a}{2^k}\rfloor}{2}\rfloor = \lfloor\frac{\frac{a}{2^k} - \{\frac{a}{2^k}\}}{2}\rfloor = \lfloor\frac{a}{2^{k+1}} - \frac{\{\frac{a}{2^k}\}}{2}\rfloor = \lfloor\lfloor\frac{a}{2^{k+1}}\rfloor + \{\frac{a}{2^{k+1}}\} - \frac{\{\frac{a}{2^k}\}}{2}\rfloor = \lfloor\frac{a}{2^{k+1}}\rfloor + \lfloor \{\frac{a}{2^{k+1}}\} - \frac{\{\frac{a}{2^k}\}}{2}\rfloor \)

Let \( \frac{a}{2^{k+1}} = n + \varepsilon, n \in \mathbb{Z}, 0 \leq \varepsilon < 1 \)

\( \implies \{\frac{a}{2^{k+1}}\} = \varepsilon \)

\( \frac{a}{2^k} = 2n + 2\varepsilon \)

\( \{\frac{a}{2^k}\} = \{ 2\varepsilon \} = 2\varepsilon - \lfloor2\varepsilon\rfloor \)

\( \frac{\{\frac{a}{2^k}\}}{2} = \varepsilon - \frac{\lfloor2 \varepsilon\rfloor}{2} \)

\( \lfloor \{\frac{a}{2^{k+1}}\} - \frac{\{\frac{a}{2^k}\}}{2}\rfloor = \lfloor\frac{\lfloor2\varepsilon\rfloor}{2}\rfloor \)

\( 0 \leq 2\varepsilon < 2 \implies 0 \leq \lfloor2\varepsilon\rfloor \leq 1 \implies 0 \leq \frac{\lfloor2\varepsilon\rfloor}{2} \leq \frac{1}{2} \implies \lfloor\frac{\lfloor2\varepsilon\rfloor}{2}\rfloor = 0 \)

\( \implies d' = \lfloor\frac{a}{2^{k+1}}\rfloor = \lfloor\frac{a}{2^{k'}}\rfloor \)

A5: obvious

Now let's prove the algorithm eventually stops

\( 1 \leq x < 2 \implies 2^p \leq a < 2^{p+1} \)

If \( a = 2^p \implies \) algorithm stops.

\( a > 2^p \implies a \geq 2^p + 1 \)

\( 1 \leq \frac{a}{2^p} < 2 \implies \lfloor\frac{a}{2^p}\rfloor = 1 \implies a - \lfloor\frac{a}{2^p}\rfloor \geq 2^p \)

This means that step L3 cannot continue indefinitely, and after at most \( p \) iterations of L3, we will get to L4.

In L4: \( a \) strictly decreases. This means L4 also cannot continue indefinitely and therefore eventually the algorithm stops
