# 1-3

Algorithm F

- ${} F1 {}$. \[ Eliminate trivial assignment \] If ${} m < n, m \leftrightarrow n {}$.
- ${} F2 {}$. \[ Find remainder \] ${} r := m \mod n {}$.
- ${} F3 {}$. \[ Is it zero? \] If ${} r = 0 {}$, terminate, answer is ${} n {}$.
- ${} F4 {}$. \[ Reduce \] ${} m \leftarrow n, n \leftarrow r {}$. Go to F2.
