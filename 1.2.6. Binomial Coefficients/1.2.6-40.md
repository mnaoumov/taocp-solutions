# 1.2.6-40

### a)

${} B(x,1) = \int_{0}^1  t^{x-1} dt = \frac{1}{x} {}$

${} B(1,x) = \int_{0}^1  (1-t)^{x-1} dt = \frac{1}{x} {}$

### b)

${} B(x+1, y) + B(x, y+1) = \int_0^1 (t^x(1-t)^y + t^{x-1}(1-t)^y)dt = {}$

${} = \int_0^1 t^{x-1}(1-t)^{y-1}(t + 1-t)dt = B(x,y) {}$

### c)

${} B(x+1, y) = \int_0^1 t^{x}(1-t)^{y-1}dt = -\frac{1}{y} \int_0^1 t^{x}d((1-t)^{y}) = -\frac{1}{y} t^{x} (1-t)^{y} \Bigr|_0^1 + \frac{x}{y} \int_0^1 t^{x-1}d(1-t)^{y}dt = \frac{x}{y} B(x,y+1) {}$

${} B(x,y) = \frac{x}{y} B(x,y+1) + B(x,y+1) = \frac{x+y}{y} B(x,y+1) {}$
