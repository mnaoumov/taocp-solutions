$\mu(x,p) = \sum_{k > 0}{\lfloor{\frac{x}{p^k}}\rfloor}$

$\binom{a+b}{a} = \frac{(a+b)!}{a!b!}$

$n = \mu(a+b,p) - \mu(a,p) - \mu(b, p)$

Let $a = \sum_{k \geq 0} a_k p^k$

$b = \sum_{k \geq 0} b_k p^k$

$a + b = \sum_{k \geq 0} c_k p^k$

[1.2.5-12](<../1.2.5. Permutations and Factorials/1.2.5-12.md>)

$\mu(a, p) = \frac{a - \sum_{k \geq 0} a_k}{p-1}$

$\mu(b, p) = \frac{b - \sum_{k \geq 0} b_k}{p-1}$

$\mu(a+b, p) = \frac{a+b - \sum_{k \geq 0} c_k}{p-1}$

$n = \mu(a+b,p) - \mu(a,p) - \mu(b, p) = \frac{\sum_{k \geq 0}(a_k + b_k - c_k)}{p-1}$

$d_0 := 0$

$c_0 = a_0 + b_0 + d_0, d_1 = 0$, if $d_0 + a_0 + b_0 < p$

$c_0 = a_0 + b_0 + d_0 - p, d_1 = 1$, if $a_0 + b_0 + d_0 \geq p$

$a_0 + b_0 - c_0 = pd_1 - d_0$

$c_1 = a_1 + b_1 + d_1, d_2 = 0$, if $a_1 + b_1 + d_1 < p$

$c_1 = a_1 + b_1 + d_1 - p, d_2 = 1$, if $a_1 + b_1 + d_1 \geq p$

$a_1 + b_1 - c_1 = pd_2 - d_1$

$\vdots$

$a_k + b_k - c_k = pd_{k+1} - d_k$

$n = \frac{\sum_{k \geq 0}(pd_{k+1} - d_k)}{p-1} = \frac{p \sum_{k \geq 1}d_k - \sum_{k \geq 0}d_k}{p - 1} = \frac{p \sum_{k \geq 0}d_k - \sum_{k \geq 0}d_k}{p - 1} = \sum_{k \geq 0}d_k = \# \{ \textrm{carries for adding}\ a+b \}$
