# 1.2.6-63

${} \sum_{j,k} (-1)^{j+k} \binom{j+k}{k+l}\binom{r}{j}\binom{n}{k}\binom{s+n-j-k}{m-j} = [x^ly^m] \sum_{j,k} (-1)^{j+k} \frac{(1+x)^{j+k}}{x^k} \binom{r}{j}\binom{n}{k} (1+y)^{s+n-j-k}y^j = {}$

${} = [x^ly^m] \sum_{j,k} \binom{r}{j} (-\frac{(1+x)y}{1+y})^j \binom{n}{k} (-\frac{1+x}{x(1+y)})^k (1+y)^{s+n} = [x^ly^m](1-\frac{(1+x)y}{1+y})^{r}(1-\frac{1+x}{x(1+y)})^n(1+y)^{s+n} = {}$

${} = [x^ly^m] \frac{(1-xy)^r}{(1+y)^r} \cdot \frac{(xy-1)^n}{x^n(1+y)^n} \cdot (1+y)^{s+n} = [x^ly^m] (-1)^n \frac{(1-xy)^{r+n}}{x^n}(1+y)^{s-r} = {}$

${} = [x^ly^m] (-1)^n \sum_k \binom{r+n}{k} (-1)^kx^ky^k \cdot x^{-n}(1+y)^{s-r} \overset{k = n + l}{=} [y^m] (-1)^l \binom{r+n}{n+l}y^{n+l}(1+y)^{s-r} =  {}$

${} = [y^m] (-1)^l \binom{r+n}{n+l}y^{n+l} \sum_k \binom{s-r}{k}y^k \overset{k = m-n-l}{=} (-1)^l \binom{r+n}{n+l}\binom{s-r}{m-n-l} {}$
