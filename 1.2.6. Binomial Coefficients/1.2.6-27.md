# 1.2.6-27

${} \sum_{k} A_k(r,t)z^k = x^r {}$

${} \sum_{k} A_k(s,t)z^k = x^s {}$

${} \sum_{n} A_n(r+s,t)z^k = x^{r+s} {}$

${} x^{r+s} = x^r x^s = \sum_{n} x^n \sum_{k} A_k(r,t)A_{n-k}(s,t) {}$

${} A_n(r+s,t) = \sum_{k} A_k(r,t)A_{n-k}(s,t) {}$

${} \sum_{k} A_k(r,t)z^k = x^r {}$

${} \sum_k \binom{s-tk}{k}z^k = \frac{x^{s+1}}{(t+1)x-t} {}$

${} \sum_{n} z^n (\sum_k A_k(r,t) \binom{s-t(n-k)}{n-k}) = \frac{x^{r+s+1}}{(t+1)x - t} = \sum_{n}\binom{r+s-tn}{n}z^n {}$

${} \sum_k A_k(r,t) \binom{s-t(n-k)}{n-k} = \binom{r+s-tn}{n} {}$
