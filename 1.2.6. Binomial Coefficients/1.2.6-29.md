# 1.2.6-29

${} (34): \sum_{k} \binom{r}{k} (-1)^{r-k} (b_0 + b_1 k + \cdots + b_r k^r) = r! b_r {}$

${} \binom{r}{k} (-1)^{r-k} = \frac{r!(-1)^{r-k}}{k!(r-k)!} = \frac{r!}{k!\cdot(-1)(-2)\cdots(-(r-k))} = \frac{r!(-1)^{r-k}}{k!(r-k)!} = \frac{r!}{(k-0)(k-1)\cdots(k-(k-1))\cdot(k-(k+1))(k-(k+2))\cdots(k-r)} = \frac{r!}{\prod_{\substack{0 \leq j \leq r\\j \neq k}}(k-j)} {}$

${} \sum_{k} \binom{r}{k} (-1)^{r-k} (b_0 + b_1 k + \cdots + b_r k^r) = \sum_{s=0}^r r! b_s \sum_{k=0}^r \frac{k^s}{\prod_{\substack{0 \leq j \leq r\\j \neq k}}(k-j)} =: A {}$

[1.2.3-33](<../1.2.3. Sums and Products/1.2.3-33.md>) ${} \implies A = \sum_{s=0}^r b_sr! \delta_{sr} = r! b_r {}$
